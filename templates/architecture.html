üîê Security & Resource Limits

Code Execution No Python code execution allowed. Only safe DSL commands executed through strict whitelist.
Command Whitelist BACKGROUND, TEXT, SHAPE, MOVE, FPS, DURATION only. No arbitrary commands permitted.
Resolution Limit Maximum 1280√ó720 pixels. Prevents resource exhaustion from massive canvases.
Duration Limit Maximum 6 seconds per video. Prevents long-running generation processes.
Frame Limit Maximum 300 frames total (24 FPS √ó 12.5 sec buffer). Bounds memory usage.
Object Limit Maximum 50 simultaneous objects. Prevents memory exhaustion from large scenes.
Color Validation Only hex format #RRGGBB accepted. Prevents injection attacks.
Coordinate Bounds All positions automatically clamped to canvas size. No out-of-bounds rendering.
Timeout Protection 30-second render timeout prevents infinite loops and hanging processes.
Secret Management API keys stored as encrypted Replit Secrets. Never exposed in logs or responses.
